The Nurse Project base module¶

The module nurse.base contains the main class nurse.base.Object from which any other class from the project derives. Also the global variable universe (nurse.base.Object) is defined in this module.

This parenthood provides objects with features for ‘low-level’ signal transmission.

The following example binds two objects via a signal/slot connection.

from nurse.base import *

class SignalListener(Object):
        def __init__(self, name):
                Object.__init__(self, name)
        def on_signal(self):
                print 'signal received'

listener = SignalListener('listener')
emitter = Object('emitter')
emitter.connect('secret_signal', listener, 'on_signal')
emitter.emit('secret_signal')

connect(signal, receiver, slot='receive_events', asynchronous=True)¶

Mimics Qt signal/slot connection.

Parameters:

signal : any pickable object
If signal is equal to ‘__all__’ the slot function will be called for any signal.
receiver : Object
Destination object.
slot : str
Name of the function to call on the receiver.
asynchronous : bool
True if the the slot is queued in the eventloop. False if the slot is called directly during the emit call.
